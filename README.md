### Sobre

Desenvolvedor **backend** e **engenheiro de IA**, construindo serviços e integrações com foco em **Python**, **FastAPI** e orquestração com **LangChain**, além de pipelines e análises em **Big Data** com **Databricks** e persistência em **MongoDB**. Também uso **TypeScript** quando o contexto do produto pede tipagem forte no ecossistema web/Node.

- APIs REST/async, integração com **LLMs (OpenAI)** e automação de fluxos inteligentes  
- Dados em volume: **Databricks**, modelagem e consumo de dados para decisão e produto  
- Interesse contínuo em **observabilidade**, boas práticas de API e código sustentável
